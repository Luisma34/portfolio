# Portofolio personal - Luis Manuel Cordón Álvarez
<p> Portfolio web profesional desarrollado como escaparate de mis proyectos y habilidades como desarrollador Full Stack. </p>

## Sobre el proyecto
<p> Diseñado con un enfoque minimalista. La web incluye una sección de proyectos, presentación personal y formulario de contacto. El código es de open source con intención de que sirva como referencia. </p>

## Stack tecnológico

### Frontend
- React + Vite
- Tailwind CSS
- Framer motion
- React Router DOM

## Estructura del proyecto

portfolio/
├── frontend/          # React + Vite
│   └── portfolio-frontend/
│       └── src/
│           ├── components/
│           ├── pages/
│           └── assets/
└── backend/           # Spring Boot
    └── luisdeveloper-api/

## Instalación y uso 
# Clonar el repositorio
git clone https://github.com/Luisma34/portfolio.git

# Instalar dependencias del frontend
cd frontend/portfolio-frontend
npm install

# Iniciar en desarrollo
npm run dev
